    Class ForwardEuler

    The class ForwardEuler is a subclass of FlowMap. It implements the forward Euler method

    \[ \mathbf{x}(n+1) = \mathbf{x}(n) + k \mathbf{f}(\mathbf{x}(n), \mathbf{u}(n)), \]

    where \(k\) is the time step size.
